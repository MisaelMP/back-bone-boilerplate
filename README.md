# Backbone Boilerplate

## Start
- Clone
- npm install
- gulp
- Open http://127.0.0.1:7000 in the browser

## Technologies used:

- [Backbone](http://backbonejs.org/)
- [Browserify](http://browserify.org/)
- [Underscore](http://underscorejs.org/)
- [Sass](http://sass-lang.com/)
- [Gulp](http://gulpjs.com/)
